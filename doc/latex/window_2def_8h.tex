\doxysubsection{Detailed Description}
Provides a definition for an arbitrarily resizeable and shiftable array of items. A window here is comprised of two ranges as defined in src/range.\+h. The \textquotesingle{}alloc\textquotesingle{} range of a window contains the total memory allocated to the window, and the \textquotesingle{}region\textquotesingle{} range of a window describes the used portion of the allocated memory.
